# ModelViewer
OSG读取模型的顶点、法向量、纹理坐标并用随机颜色绘制出来
# 简介
此程序是在使用OSG的过程中，需要获取模型的顶点、法向、纹理坐标等信息，然后使用不同的颜色绘制出来，以区别不同模型。为了知道模型的名称，在模型的周围增加了模型的名称显示。效果如下：
